# @amlplugins/google-search-console

Native `@googleapis/searchconsole` SDK re-exported under the `@amlplugins` namespace.

> Search Console (searchconsole_v1).

## Install

```bash
npm install @amlplugins/google-search-console
```

## Use

```ts
import { /* named exports from @googleapis/searchconsole */ } from "@amlplugins/google-search-console";
// or
import sdk from "@amlplugins/google-search-console";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
